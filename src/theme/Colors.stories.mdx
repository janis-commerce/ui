# Color :

Por el momento, por cuestiones de retrocompatibilidad con los proyectos, contamos con dos paletas de colores, las cuales tienen distintas estructuras pero comparten algunos valores.
Si bien se puede importar cualquiera de estas dos paletas ya mencionadas, utilizando la funcion `getColor`, podemos acceder a cualquiera de estas dos paletas.
La busqueda de un valor dependera de los parametros recibidos.

## implementaci√≥n getColor :

Para obtener un color se puede utilizar la funcion `getColor` de distintas maneras.

- Con un solo parametro que sea correspondiente a un color de `views palette`.
- Con un parametro que corresponda a un grupo de `mainPallet`, por default devolvera el valor encontrado `${group}.main`
- Armando el nombre con la ubicacion correspondiente del color. por ejemplo `${group}.${color}.
- Utilizando como primer parametro el grupo y segundo el color.

```js
import { getColor } from 'theme/utils';

const viewsGreen = getColor('green');

const alertYellow = getColor('alert');

const alertYellowHover = getColor('alert.hover');

const errorDark = getColor('error', 'dark');
```
