# Icon

Los iconos son simples svgs que internamente se arman en base a un [path](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/path).

Para renderizar un icono solo se necesita pasar el <Link kind="theme-icons--icons">nombre</Link> como prop.
